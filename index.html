GitHub Essentials — One‑Page Cheat Sheet

1. Repository Basics

Create new repo:
git init
git remote add origin <url>

Clone existing repo:
git clone <url>

Add README:
touch README.md

2. Staging & Committing

Stage all changes:
git add .

Stage specific file:
git add <file>

Commit:
git commit -m "message"

Unstage file:
git restore --staged <file>

Undo last commit (keep changes):
git reset --soft HEAD~1

Undo last commit (discard changes):
git reset --hard HEAD~1

3. Branching

List branches:
git branch

Create branch:
git branch <name>

Switch branch:
git checkout <name>

Create + switch:
git checkout -b <name>

Delete branch:
git branch -d <name>

4. Syncing & Merging

Pull latest:
git pull origin <branch>

Push changes:
git push origin <branch>

Set upstream:
git push -u origin <branch>

Merge branch into current:
git merge <branch>

5. Fixing Mistakes

Restore file to last commit:
git restore <file>

Revert commit safely:
git revert <commit-hash>

View history:
git log --oneline --graph --decorate

6. GitHub Workflow (UI)

Create Pull Request
Add description
Request review
Merge
Delete branch

7. Issues & Labels

Track bugs, tasks, enhancements
Use labels: bug, feature, urgent, etc
Use milestones for releases
